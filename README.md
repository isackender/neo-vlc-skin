# Neo VLC
A 1bit distractionless skin for VLC media player.

![Demo image](https://i.imgur.com/fTZ20nh.png)

![Demo image](https://i.imgur.com/s9TG5NG.png)

![Demo image](https://i.imgur.com/e8GXtFa.png)

![Demo image](https://i.imgur.com/0UAlYeb.png)

You can download the project to build your own .vlt file and open it in VLC media player, or you can modify it to your needs with VLC media player [Skin Editor](https://images.videolan.org/vlc/skineditor.html)
